# Python 全栈工程师转型之路

本项目旨在帮助你从一名前端全栈工程师转型为 Python 全栈工程师，并掌握使用 Python 进行 AI 相关工作的技能，特别是 RAG（检索增强生成）技术。

## 学习路线规划

我们将按照以下阶段进行学习：

1. **巩固 Python 基础**
2. **掌握 Python 后端开发框架**
3. **深入学习数据库与缓存**
4. **学习前端基础（复习）**
5. **实战项目练习**
6. **AI 与 RAG 技术入门**
7. **RAG 实战项目**

每个阶段都会有相应的学习资源、实践项目和检查点，确保你能循序渐进地掌握所需技能。
