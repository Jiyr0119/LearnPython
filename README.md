# Python 全栈工程师转型之路

本项目旨在帮助你从一名前端全栈工程师转型为 Python 全栈工程师，并掌握使用 Python 进行 AI 相关工作的技能，特别是 RAG（检索增强生成）技术。

## 学习路线规划

我们将按照以下优化的阶段进行学习，注重现代Python开发实践、测试、安全性和性能：

1. **巩固 Python 基础** (强化类型提示、测试、代码质量工具)
2. **Python 异步编程**
3. **掌握 Python 后端开发框架** (FastAPI为主，注重安全、性能、测试)
4. **深入学习数据库与缓存** (包含优化、安全、性能)
5. **学习前端基础（复习）** (重点与AI后端交互)
6. **生产部署与 DevOps** (提前，包含Docker、CI/CD、监控)
7. **实战项目练习** (完整全栈应用)
8. **AI 与 RAG 技术入门**
9. **RAG 实战项目** (全栈+AI应用)

**补充资源**:
- **全栈开发技能提升指南**: 详细讲解前后端类型一致性、API设计、微服务架构等全栈核心概念
- **Python专家代理配置**: 为有前端背景的开发者定制的AI助手配置

每个阶段都会有相应的学习资源、实践项目和检查点，确保你能循序渐进地掌握所需技能。优化后的路径更注重现代Python开发实践、测试驱动开发、安全性、性能优化和生产部署。
